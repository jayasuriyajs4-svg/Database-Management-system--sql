SQL Database Management and Analytics Experiments

Overview

This repository contains a collection of SQL and Database Management System (DBMS) experiments implemented using MySQL. The experiments demonstrate fundamental and advanced database concepts, including schema design, SQL queries, subqueries, joins, triggers, stored procedures, transactions, indexing, and data analysis.

Objectives

- Design and implement relational databases.
- Perform data retrieval and analysis using SQL queries.
- Apply normalization and database design principles.
- Implement stored procedures, triggers, and transactions.
- Analyze sales and customer data using SQL techniques.
- Optimize query performance using indexes.

---

Experiments Included

1. Database Design and Organization Management System

- Database creation and schema design.
- Table creation and normalization concepts.
- Implementation of primary and foreign keys.

2. Academic Management System

- Faculty, Student, and Department management.
- Referential integrity and foreign key constraints.
- Department assignment and relationship management.

3. Sales Data Analysis Using GROUP BY

- Aggregate functions.
- GROUP BY operations.
- Sales trend analysis.

4. Sales and Customer Data Analysis Using SQL Subqueries

- Correlated subqueries.
- Nested queries.
- Customer purchase analysis.

5. SQL Joins on Sales and Customer Tables

- INNER JOIN.
- LEFT JOIN.
- Data integration across multiple tables.

6. SQL Triggers for Stock Management

- Automatic stock updates.
- Trigger implementation.
- Inventory management operations.

7. SQL Stored Procedures

- Procedure creation.
- Data insertion and updates.
- Reusable database operations.

8. Commit and Rollback Operations

- Transaction management.
- Data consistency.
- Error recovery mechanisms.

9. Database Indexing and Query Optimization

- Index creation.
- Duplicate key updates.
- Efficient data retrieval.
